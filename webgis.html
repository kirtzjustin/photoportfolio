My Travel Map

As I travel to new places I log my location into Google Maps Saved Places and then periodically convert them to a shapefile for use in this GIS web application. I've also logged the location of some of my favorite personal photographs. I developed this app using the ESRI JavaScript API Version 4.31.

Libraries: ESRI JavaScript API 4.x, ArcGIS Online, Bootstrap, jQuery, Chart.js, Tabulator

After growing up in Charlotte, North Carolina I decided to get a change of scenery from city life and attended Appalachian State University in rural Boone, North Carolina. Having a life-long interest in science, I majored in Geography with a concentration in GIS (Geographic Information Systems). Upon graduating college in 2010 I served as the GIS Planner for the City of Greer, SC where I was a one man GIS department. During this time I discovered a passion for writing code and taught myself Python and JavaScript in my spare time at home. I oversaw the implementation and development of Greer"s web GIS platform using ArcGIS Server and wrote Greer"s custom web-based GIS viewer using the ESRI JavaScript 4.x API. After seven years with Greer I decided to transition to the private sector and joined SynTerra Corporation in downtown Greenville, SC as the GIS Administrator specializing in GIS web development. After three years with SynTerra I decided to return to the public sector as the GIS Manager with Greenville Water, where I lead the GIS Department.
